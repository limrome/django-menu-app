# Django Menu App

Приложение для древовидного меню на Django с поддержкой неограниченной вложенности.

## Функции

- **Древовидная структура** - неограниченная вложенность пунктов меню
- **Один запрос к БД** - каждое меню загружается одним оптимизированным запросом
- **Автоматическое определение активного пункта** - на основе текущего URL страницы
- **Умное раскрытие** - родители активного пункта и их непосредственные дети развернуты автоматически
- **Несколько меню на странице** - разные меню в разных местах страницы
- **Кэширование** - для максимальной производительности
- **Поддержка named URL** - можно использовать как явные URL, так и именованные паттерны из urls.py
